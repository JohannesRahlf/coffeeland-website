{{- $dest := .Destination -}} {{- $isRemote := or (hasPrefix $dest "http://") (hasPrefix $dest "https://") (hasPrefix $dest "//") (hasPrefix $dest "data:") -}} {{- $src := $dest -}} {{- if not $isRemote -}} {{- $localDest := $dest -}} {{- if hasPrefix $localDest "/" -}} {{- $localDest = strings.TrimPrefix "/" $localDest -}} {{- end -}} {{- $src = absURL $localDest -}} {{- end -}}
